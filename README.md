## Dependencias y como correr

Se puede usar Google Colab o usar un entorno virtual.

```bash
pip install virtualenv
```

Con venv

```bash
python -m venv venv
source venv/bin/activate
```

Dentro del entorno virtual:

```bash
pip install jupyterlab
```

Levanta el servidor:

```bash
jupyter lab
```
